# MP1.9 Electron Microscopy Structure Validation

Validation that dual-axis models reproduce electron microscopy observed interface structure.

## Repository role

- GitHub repo: `alexanderholman/mp1-9-em-structure-validation`
- Program: MP1 poly-axis interface analysis in Si
- Scope: Micro-paper execution unit.

## Getting started

- Read `plan.md` (micro) or `plan_micro_papers.md` (macro).
- Review `tasks/wip.md` and `tasks/triage.md` for current execution state.
- Follow setup notes in `INSTALL.md`.

## AgentFactory

This repository keeps the AgentFactory structure (`agents/`, `agents.yaml`, `specs.md`, `decisions.md`, `agent_runs.md`) for traceable AI-assisted workflows.
