# Installation

## Prerequisites

- Git
- Python 3.11+
- Optional: LaTeX distribution for manuscript builds

## Clone

```bash
git clone git@github.com:alexanderholman/mp1-9-em-structure-validation.git
cd mp1-9-em-structure-validation
```

## Update

```bash
git pull --ff-only
```

## Remove

Delete the cloned directory when no longer needed.
